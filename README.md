# Programming for Industrial Applications

A Python code collection with exercises for Programming for Industrial Applications.

## Key Concepts Covered

- Input/output (`input()`, `print()`, f-strings)  
- Data types and arithmetic operations  
- Conditionals (`if`, `elif`, `else`)  
- Loops (`for`, `while`, `break`, `continue`)  
- Functions (defining and using functions)  
- Random numbers (`random.randint()`)  
- Problem-solving (pairs, leftovers)  
- String formatting and grammar handling  
- File organization and code structure
